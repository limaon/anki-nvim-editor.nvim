# anki-nvim-editor

Plugin para Neovim que permite editar templates de cartas do Anki diretamente no editor.

## Status: Em Desenvolvimento

Este plugin está em fase de desenvolvimento inicial. Veja [PLANO.md](./PLANO.md) para detalhes do roadmap.

## Requisitos

- Neovim 0.6+
- Anki desktop instalado
- [Anki-Connect](https://ankiweb.net/shared/info/2055492159) add-on instalado no Anki
- Anki rodando com Anki-Connect ativo
